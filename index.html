Создание файла презентации.

1. Наводим курсор мыши на свободное от иконок место рабочего стола, кликаем(нажимаем) правую кнопку мыши, в появившемся меню выбираем "Создать", в раскрывшемся списке выбрать "Презентация Microsoft PowerPoint".

2. Создаём файл презентации Microsoft PowerPoint, файл имеет расширение *.pptx или *.ptt, где *(звездочка) имя файла, по умолчанию , при первом создании имеет имя "Презентация Microsoft PowerPoint".

3. Открываем файл "Презентация..." двойным кликом левой кнопки мыши.

4. Создаём слайд, это можно сделать тремя способами:

- на вкладке "Главная" выбираем " Создать слайд",

- на клавиатуре нажимаем "CTRL+M";

- на пустом месте слева, наводим курсор мыши и кликаем правую кнопку, выбираем "Создать слайд".

5. Изменяем фон слайда --> переходим на вкладку "Дизайн", выбираем "Формат фона", в появившемся окошке смотрим, чтоб было отмечено "Сплошная заливка".

Меняем цвет заливки - строка "Цвет", кнопка с ведерком, наводим курсор мыши, кликаем левой кнопкой мыши, из выпадающего набора цветов, выбираем "черный".

Добавление картинки.

Итак, имеем слайд презентации, с чёрным фоном (нам чёрный фон нужен для "невидимости":-). Для того, что бы вставить картинку:
1. Мы берём и "перетаскиваем" её, хочу заметить это один из способов вставить картинку.

2. Растягиваем или сжимаем картинку за уголок. Центруем её.

3. Сохраняем файл. Это можно сделать либо нажав "CTRL+S", либо иконка "Дискета", либо зайти на вкладку "Файл", потом выбрать "Сохранить" или "Сохранить как"(в этом случае мы можем переименовать файл и указать иное место для сохранения).

Запуск слайда.

Для запуска слайда нужно нажать на иконку "Начать сначала"

или "Слайд-шоу".
